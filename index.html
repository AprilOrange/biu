Biu

Basic Usage

biu('text you want to show')
biu('success', 'text with a type')
biu('info', 'do not auto hide me', false)

Advanced

// only `text` is required
// everything else uses the default value if not set
biu({
  type: 'info',
  text: 'some message',
  height: '50px',
  lineHeight: '50px',
  top: '-55px',
  closeButton: 'x',
  autoFade: true
})
